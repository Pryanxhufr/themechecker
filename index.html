BNB Wallet Address Detection Methods

Important: These methods only work if the user has previously connected their wallet or granted permissions. They respect user privacy and won't force connections.

Method 1: Check Already Connected Accounts

Checks if MetaMask/wallet is already connected without prompting

Method 2: Check Injected Wallet

Detects if wallet extension is installed and checks current account

Method 3: BNB Smart Chain Specific Check

Switches to BSC network and checks for connected accounts

Method 4: Silent Permission Check

Uses eth_accounts (doesn't trigger popup if not permitted)

If No Address Found - Connect Wallet

Only shows connection prompt when needed
